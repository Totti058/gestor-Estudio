# gestor-Estudio
Aplicación web para organizar apuntes de estudio. Permite registrar notas, seleccionar materia e importancia y mostrarlas en una tabla dinámica. Desarrollada en HTML, CSS y JavaScript, enfocada en práctica del DOM y diseño responsive.
Características
- Guarda datos en LocalStorage para persistencia
- Interfaz dinámica con JavaScript
- Compatible en dispositivos móviles
